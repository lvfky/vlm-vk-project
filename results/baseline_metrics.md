# Метрики базовой модели (zero-shot, до дообучения)

Ноутбук `03`. Модель `deepvk/llava-gemma-2b-lora` без дообучения.

| Бенчмарк | n | Метрика | Значение |
|---|---|---|---|
| GQA-ru | 1000 | accuracy (exact-match) | **0.466** |
| MMBench-ru | 1000 | accuracy (буква A/B/C/D) | **0.642** |

## Детали оценки

- Выборка перемешана с `seed=42` перед взятием первых N — чтобы убрать перекос
  (в MMBench первые примеры сгруппированы по ответу, а модель склонна отвечать «A»).
  То же N и seed нужно использовать при оценке ПОСЛЕ обучения — для честного сравнения.
- **GQA**: промпт «вопрос + 'Ответь одним словом.'», ответ нормализуется (нижний регистр,
  без пунктуации) и сравнивается с эталоном (точное совпадение, с допуском на многословный ответ).
- **MMBench**: промпт «вопрос + варианты A/B/C/D + 'Ответь только буквой'», из ответа
  извлекается буква; варианты со значением `nan` отбрасываются.

Эти числа — точка отсчёта. После LoRA-дообучения повторяем оценку с тем же N и seed
и строим таблицу/график «до/после».
