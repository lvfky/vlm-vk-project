# Разведка данных (ноутбук 01)

Краткие итоги осмотра трёх датасетов из коллекции `deepvk`. Смотрел через
`load_dataset_builder` (метаданные) + `streaming=True` (первые примеры), чтобы не качать
десятки ГБ на диск Colab.

## deepvk/LLaVA-Instruct-ru — данные для дообучения

- Конфиг: `default`. Сплиты: train 109 905 + validation 34 075 = **143 980** примеров.
- download_size ≈ 274 МБ, dataset_size ≈ 239 МБ (это только текст, без картинок).
- Поля: `conversations` (список реплик human/gpt, в реплике человека есть токен `<image>`),
  `type`, `id`, `image`.
- **Важно:** поле `image` — это строка-путь вида `coco/train2017/000000253464.jpg`,
  а не картинка. Сами изображения — из **COCO train2017**, в датасете их нет.
  → для обучения нужно отдельно достать нужные картинки COCO (весь набор ~18 ГБ,
  на free-Colab тянуть только подвыборку по путям/id).

## deepvk/GQA-ru — бенчмарк 1

- Датасет разбит на конфиги: `*_images` (id + картинка) и `*_instructions` (вопросы/ответы).
  Части джойнятся: `instructions.imageId` == `images.id`.
- Оцениваем на `testdev`: **398 картинок** (`testdev_balanced_images`) +
  **12 216 вопросов** (`testdev_balanced_instructions`).
- Нужные поля вопроса: `imageId`, `question` (рус.), `answer` (короткий эталон, напр. «нет», «женщины»).
  Остальное (fullAnswer, semantic, groups, types) — исходная разметка GQA, не нужна.
- Картинки встроены (Image feature, decode=True).
- Метрика: **exact-match accuracy** по `answer` с нормализацией (нижний регистр, убрать пунктуацию/пробелы).

## deepvk/MMBench-ru — бенчмарк 2

- Конфиг `default`, сплит **dev: 3 910** примеров.
- Поля: `index`, `question`, `hint`, `A`, `B`, `C`, `D`, `answer` (буква A/B/C/D),
  `category`, `l2-category`, `image` (встроена), `source`, `comment`, `split`.
- Лишние варианты = `'nan'` (вопрос может иметь 2–4 варианта) — при построении промпта пропускать.
- В сплите `dev` ответы есть → метрику считаем локально.
- Метрика: **accuracy** по выбранной букве.

## Выводы для дальнейшей работы

1. Оба бенчмарка самодостаточны (картинки + ответы внутри) → zero-shot оценку можно делать сразу.
2. Для дообучения на LLaVA-Instruct-ru потребуется отдельно подтянуть изображения COCO train2017
   (экономно — только нужную подвыборку).
