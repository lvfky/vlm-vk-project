# VLM на открытых данных VK

Дообучение Vision-Language модели на открытых данных коллекции [`deepvk`](https://huggingface.co/deepvk) и оценка на русскоязычных бенчмарках **GQA-ru** и **MMBench-ru**.

## Цель и задачи

**Цель.** Получить как можно более высокую метрику на бенчмарках `GQA-ru` и `MMBench-ru`, дообучив открытую VLM на открытых данных VK.

**Задачи.**
1. Настроить окружение (Google Colab, GPU T4, 16 ГБ VRAM).
2. Изучить датасеты: структура, примеры, размеры.
3. Взять за базу `deepvk/llava-gemma-2b-lora`, проверить инференс и влезаемость QLoRA (4-bit) в 16 ГБ.
4. Zero-shot оценка базовой модели на `GQA-ru` / `MMBench-ru`.
5. LoRA-дообучение на подвыборке `deepvk/LLaVA-Instruct-ru`.
6. Повторная оценка, сравнение метрик до/после.

## Ограничения окружения

- Обучение и инференс — только в **Google Colab** (бесплатный тариф, T4, 16 ГБ VRAM, лимит сессии ~12 ч с риском обрыва).
- Дообучение — через **LoRA/QLoRA** (4-bit, `bitsandbytes`): полное дообучение в T4 не помещается.
- Чекпоинты сохраняются в Google Drive для возможности resume после обрыва сессии.

## Используемые ресурсы (HuggingFace, коллекция `deepvk`)

| Тип | ID | Назначение |
|---|---|---|
| Модель (основная) | `deepvk/llava-gemma-2b-lora` | VLM на базе Gemma-2b, дообучена LoRA в LLaVA-сетапе |
| Модель (запасная) | `deepvk/llava-saiga-8b` | VLM на базе Saiga-8b, тяжелее |
| Данные для дообучения | `deepvk/LLaVA-Instruct-ru` | ~144k инструктивных примеров |
| Бенчмарк 1 | `deepvk/GQA-ru` | ~80k примеров |
| Бенчмарк 2 | `deepvk/MMBench-ru` | ~3.9k примеров |

## Как запускать

Ноутбуки из `notebooks/` открываются в Google Colab (`File → Open notebook → GitHub/Upload`).
Для скачивания моделей и датасетов нужен HuggingFace-токен с **read**-доступом — логин выполняется
ячейкой `notebook_login()` в начале каждого ноутбука.
